## A propos

Ce projet a pour but de pouvoir créer des évènements, les rejoindre et de pouvoir créer dans ces évènements des listes avec des idées de cadeaux dedans.

Il est nécessaire de se créer un compte pour pouvoir accéder au reste de l'application.

Après s'être connecté, on arrive sur la page des évènements et deux choix sont possibles :
	- Créer un évènement
	- Rejoindre un évènement via le code de l'évènement

Une fois dans l'évènement, chacun peut créer sa liste et mettre dedans ces cadeaux.

Seul la personne qui à créer la liste ne peut pas voir les idées cochées dedans.

Toutes les autres personnes peuvent cochés une idée.

Une idée cochée et enregistrée ne peut pas être décoché.
